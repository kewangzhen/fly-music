## 7. 项目结构

### 7.1 后端目录结构
```
src/main/java/com/example/flymusic/
├── controller/        # 控制器
├── service/           # 服务层
├── repository/        # 数据访问层
├── entity/            # 实体类
├── dto/               # 数据传输对象
├── config/            # 配置类
├── utils/             # 工具类
├── security/          # 安全相关
├── recommendation/    # 推荐算法
├── ai/                # AI相关功能
├── social/            # 社交功能
└── admin/             # 管理员功能
```

### 7.2 前端目录结构
```
src/
├── components/        # 组件
├── views/             # 页面
├── router/            # 路由
├── store/             # 状态管理
├── services/          # API服务
├── utils/             # 工具类
├── assets/            # 静态资源
├── ai/                # AI相关功能
├── social/            # 社交功能
└── admin/             # 管理员功能
```
