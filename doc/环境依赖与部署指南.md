# Fly Music 环境依赖与部署指南

> 本文档记录 Fly Music 项目所需的环境依赖及 AI 模型部署步骤

---

## 一、环境依赖

### 1.1 后端环境

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| JDK | 17+ | Java 运行环境 |
| Maven | 3.8+ | 项目构建工具 |
| MySQL | 8.0+ | 数据库 |
| Redis | 6.0+ | 缓存（可选） |

**Windows 安装：**
```powershell
# 使用 Winget 安装
winget install Oracle.JDK.17
winget install Apache.Maven
winget install MySQL.MySQL
```

**环境变量配置：**
```powershell
# JDK 环境变量
# JAVA_HOME = C:\Program Files\Java\jdk-17
# 在 Path 中添加: %JAVA_HOME%\bin

# Maven 环境变量
# MAVEN_HOME = C:\apache-maven
# 在 Path 中添加: %MAVEN_HOME%\bin

# 验证安装
java -version
mvn -version
```

### 1.2 前端环境

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| Node.js | 18+ | 前端运行环境 |
| npm | 9+ | 包管理器 |
| pnpm | 8+ | 推荐使用 |

**Windows 安装：**
```powershell
# 使用 Winget 安装 Node.js
winget install OpenJS.NodeJS.LTS

# 安装 pnpm
npm install -g pnpm
```

**环境变量配置：**
```powershell
# Node.js 环境变量（通常自动配置）
# 在 Path 中添加: C:\Program Files\nodejs

# 验证安装
node -v
npm -v
pnpm -v
```

### 1.3 AI 模型服务环境

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| Python | 3.10+ | AI 服务运行环境 |
| CUDA | 12.1+ | GPU 加速（可选） |
| PyTorch | 2.0+ | 深度学习框架 |

**Windows 安装：**
```powershell
# 安装 Python 3.11
winget install Python.Python.3.11
```

**环境变量配置：**
```powershell
# Python 环境变量
# 在安装时勾选 "Add Python to PATH"
# 或手动配置:
# PYTHON_HOME = C:\Python311
# 在 Path 中添加: %PYTHON_HOME%;%PYTHON_HOME%\Scripts

# 安装 PyTorch (CPU 版本)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu

# 安装 PyTorch (GPU 版本，需要 NVIDIA CUDA)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

**验证安装：**
```powershell
python --version
pip --version
import torch
print(torch.__version__)
print(torch.cuda.is_available())  # GPU 版本才为 True
```

## 二、本地运行

### 2.1 后端启动

```powershell
# 1. 进入项目目录
cd fly-music

# 2. 配置数据库连接（修改 src/main/resources/application.properties）
# spring.datasource.url=jdbc:mysql://localhost:3306/fly_music
# spring.datasource.username=root
# spring.datasource.password=your_password

# 3. 启动后端
mvn spring-boot:run
```

后端启动成功后访问：http://localhost:8080

### 2.2 前端启动

```powershell
# 1. 进入前端目录
cd frontend

# 2. 安装依赖
pnpm install
# 或 npm install

# 3. 启动开发服务器
pnpm dev
```

前端启动成功后访问：http://localhost:5173

## 三、Windows 上部署 AI 模型

### 3.1 支持的 AI 模型

| 模型 | 功能 | 最低显存 | 推荐显存 |
|------|------|----------|----------|
| MusicGen | 音乐生成 | 8GB | 16GB |
| ACE-Step | 音乐生成+曲风识别 | 4GB (量化) | 16GB |

### 3.2 安装 ACE-Step（Windows）

#### 步骤 1：安装 CUDA（可选）

```powershell
# 下载 CUDA Toolkit 12.1
# 访问: https://developer.nvidia.com/cuda-downloads
# 选择: Windows -> x86_64 -> exe(local)

# 验证安装
nvidia-smi
nvcc --version
```

#### 步骤 2：安装 Python 依赖

```powershell
# 创建虚拟环境
python -m venv venv
venv\Scripts\activate

# 安装 PyTorch (GPU 版本)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# 安装 transformers 和其他依赖
pip install transformers diffusers gradio scipy soundfile loguru einops accelerate fastapi
```

#### 步骤 3：下载 ACE-Step 模型

```powershell
# 安装 HuggingFace CLI
pip install huggingface-hub

# 登录 HuggingFace（需要账号）
huggingface-cli login

# 下载模型
huggingface-cli download ACE-Step/ace-step-1.5-base --local-dir ./models/ace-step-1.5-base
```

#### 步骤 4：启动 ACE-Step API

```powershell
# 克隆 ACE-Step 仓库
git clone https://github.com/ACE-Step/ACE-Step-1.5.git
cd ACE-Step-1.5

# 安装依赖
pip install -r requirements.txt

# 启动 API 服务
python -m acestep.api_server --port 8001
```

### 3.3 使用 Docker 部署（推荐）

如果本地环境配置困难，可以使用 Docker 快速部署：

```powershell
# 1. 安装 Docker Desktop for Windows
# 访问: https://www.docker.com/products/docker-desktop/

# 2. 创建 Dockerfile
# 在项目根目录创建 Dockerfile.ai

# 3. 构建镜像
docker build -f Dockerfile.ai -t fly-music-ai .

# 4. 运行容器
docker run -d -p 5001:5001 --gpus all fly-music-ai
```

### 3.4 配置 Fly Music 连接 AI 服务

修改 `src/main/resources/application.properties`：

```properties
# AI 音乐服务配置
ai.music.service.url=http://localhost:5001
ai.music.enabled=true
```

## 四、常见问题

### 4.1 CUDA 相关

**Q: 显示 "CUDA not available"**
A: 检查 PyTorch 是否安装 CUDA 版本：
```python
import torch
print(torch.cuda.is_available())
print(torch.version.cuda)
```

**Q: 显存不足**
A: 使用量化版本或减少 batch size：
```python
# 量化推理
model = model.to(dtype=torch.float16)
```

### 4.2 Python 环境

**Q: pip 安装失败**
A: 建议使用 conda 或 venv：
```powershell
# 使用 conda
conda create -n fly-music python=3.11
conda activate fly-music

# 使用 venv
python -m venv venv
venv\Scripts\activate
```

### 4.3 Windows 兼容性

**Q: 模块导入失败**
A: 确保使用 Visual C++ Build Tools：
```powershell
# 安装 Visual C++ Build Tools
# 访问: https://visualstudio.microsoft.com/visual-cpp-build-tools/
```

## 五、性能优化

### 5.1 GPU 推理优化

| 优化项 | 说明 |
|--------|------|
| FP16 量化 | 减少 50% 显存，提升 2x 速度 |
| TensorRT | 提升 3-5x 推理速度 |
| vLLM | 提升批量生成效率 |

### 5.2 CPU 推理配置

如果使用 CPU 运行：

```python
# 设置 CPU 线程数
import torch
torch.set_num_threads(8)

# 使用 quantized 模型
model = model.to(dtype=torch.float16)
```
