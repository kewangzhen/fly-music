# 🎵 听歌报告功能设计文档

## 一、需求概述

听歌报告是一个基于用户历史行为数据的个性化年度报告功能，参考网易云音乐年度报告设计，通过数据可视化展示用户的音乐品味和听歌习惯。

### 核心特性
- ✅ 基于用户历史行为数据
- ✅ 实时生成（按需计算）
- ✅ 多维度数据统计
- ✅ 数据可视化展示

---

## 二、报告内容设计

### 2.1 基础统计

| 指标 | 说明 |
|------|------|
| 听歌曲数 | 用户累计播放过的歌曲总数 |
| 听歌时长 | 累计播放时长（小时） |
| 听歌天数 | 累计听歌的天数 |
| 播放次数 | 累计播放总次数 |

### 2.2 音乐流派分布（饼图）

统计用户播放歌曲的分类分布，生成饼图数据：
```
例如：
- 流行: 35%
- 摇滚: 25%
- 民谣: 20%
- 电子: 12%
- 其他: 8%
```

### 2.3 最喜欢的歌曲 TOP10

按播放次数排序的前10首歌曲：
```json
[
  {"rank": 1, "title": "孤勇者", "artist": "陈奕迅", "playCount": 156},
  {"rank": 2, "title": "起风了", "artist": "买辣椒也用券", "playCount": 142},
  ...
]
```

### 2.4 最喜欢的歌手

按播放次数统计的歌手排行：
```json
[
  {"rank": 1, "name": "周杰伦", "songCount": 45, "playCount": 320},
  {"rank": 2, "name": "林俊杰", "songCount": 30, "playCount": 256},
  ...
]
```

### 2.5 经常听的标签（词云图）

用户偏好标签及分数，用于生成词云：
```json
{
  "摇滚": 450,
  "民谣": 220,
  "流行": 180,
  "电子": 120,
  "古典": 80
}
```

### 2.6 听歌时间分布

按小时统计听歌时间分布：
```json
{
  "00:00-06:00": 50,
  "06:00-12:00": 120,
  "12:00-18:00": 280,
  "18:00-24:00": 450
}
```

## 三、数据库设计

### 3.1 听歌报告表（可选，缓存用）

```sql
CREATE TABLE IF NOT EXISTS `listening_report` (
    `id` BIGINT NOT NULL AUTO_INCREMENT,
    `user_id` BIGINT NOT NULL,
    `report_year` INT NOT NULL COMMENT '报告年份',
    `total_songs` INT DEFAULT 0 COMMENT '听歌曲数',
    `total_hours` DECIMAL(10,2) DEFAULT 0 COMMENT '听歌时长(小时)',
    `total_days` INT DEFAULT 0 COMMENT '听歌天数',
    `total_plays` BIGINT DEFAULT 0 COMMENT '播放次数',
    `category_distribution` JSON COMMENT '分类分布',
    `top_songs` JSON COMMENT 'TOP歌曲',
    `top_artists` JSON COMMENT 'TOP歌手',
    `tag_distribution` JSON COMMENT '标签分布',
    `time_distribution` JSON COMMENT '时间分布',
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_user_year` (`user_id`, `report_year`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='听歌报告表';
```

## 五、算法设计

### 5.1 听歌时长计算

```java
// 假设每首歌平均时长为 song.duration 秒
// 播放次数 × 平均时长 = 播放总时长
totalHours = (playCount × averageDuration) / 3600
```

### 5.2 分类分布计算

```java
// 遍历用户播放历史，统计每个分类的播放次数
Map<Category, Long> categoryCount = new HashMap<>();
for (PlayHistory history : histories) {
    Category category = history.getSong().getCategory();
    categoryCount.merge(category, 1L, Long::sum);
}

// 转换为百分比
total = sum(categoryCount.values());
percentage = categoryCount.get(cat) / total × 100%
```

### 5.3 TOP歌曲计算

```java
// 按歌曲ID分组，统计播放次数
Map<Long, Long> songPlayCount = new HashMap<>();
for (PlayHistory history : histories) {
    songPlayCount.merge(history.getSongId(), 1L, Long::sum);
}

// 排序取前10
List<Map.Entry<Long, Long>> sorted = songPlayCount.entrySet().stream()
    .sorted(Map.Entry.comparingByValue().reversed())
    .limit(10)
    .collect(Collectors.toList());
```
