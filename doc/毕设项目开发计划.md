# 🎓 Fly Music 毕设项目开发计划

## 项目概述

- **项目名称**: Fly Music 音乐推荐平台
- **技术栈**: Spring Boot 3.2.0 + Vue 3 + MySQL
- **项目类型**: 前后端分离的全栈应用
- **答辩时间**: 2025年5月
