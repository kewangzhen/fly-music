## 附录：数据字典

### 行为类型 (behavior_type)
| 值 | 描述 | 权重 |
|---|------|------|
| 1 | 播放 | +1 |
| 2 | 完整收听 | +2 |
| 3 | 收藏 | +5 |
| 4 | 分享 | +3 |
| 5 | 下载 | +3 |
| 6 | 评论 | +2 |

### 收藏目标类型 (target_type)
| 值 | 描述 |
|---|------|
| 1 | 歌曲 |
| 2 | 歌单 |
| 3 | 专辑 |

### 榜单类型 (ranking_type)
| 值 | 描述 |
|---|------|
| global | 全局热门榜 |
| category | 分类热门榜 |
| new | 新歌榜 |
| soaring | 飙升榜 |

### AI生成状态 (status)
| 值 | 描述 |
|---|------|
| 0 | 生成中 |
| 1 | 成功 |
| 2 | 失败 |
