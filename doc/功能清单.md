# 基于SpringBoot的个性化音乐推荐系统功能清单
## 一、系统整体角色
本系统包含两大核心角色，分别是**用户**与**管理员**，不同角色对应不同功能权限与操作模块。

## 二、用户功能模块
### （一）用户登录注册
1. 用户注册
2. 用户登录
3. 密码找回
4. 个人信息修改
5. 注册会员

### （二）音乐浏览搜索
1. 音乐列表浏览
2. 音乐搜索
3. 音乐详细信息查看
4. 歌手信息查看

### （三）音乐播放与创作
1. 音乐播放
2. 播放列表管理
3. 播放历史查看
4. 收藏音乐
5. 原创音乐上传
6. 上传记录管理

### （四）个性化推荐
1. 心动歌单
2. 分类推荐
3. 雷达榜单
4. 热门榜单
5. 听歌报告

### （五）AI音乐实验室
1. AI文本生成音乐
2. AI听歌识别曲风

### （六）社交互动
1. 关注用户
2. 粉丝查看
3. 动态发布
4. 点赞评论

## 三、管理员功能模块
### （一）用户管理
1. 用户信息查看
2. 用户状态管理
3. 用户权限设置
4. 用户数据分析
5. VIP会员监控

### （二）音乐库管理
1. 音乐上传
2. 音乐信息编辑
3. 音乐分类管理
4. 音乐审核

### （三）推荐决策看板
1. 听歌大数据
2. 推荐权重设置

### （四）系统管理
1. 系统配置管理
2. 操作日志查看
3. 数据备份恢复
4. 系统监控
